# How to Run

1.  Open terminal in this directory.
2.  Run `npm run dev`.
3.  If port 3000 is in use, Next.js will try another port or fail. You can force a port:
    `npm run dev -- -p 3005`
4.  Open the URL shown in the terminal (usually http://localhost:3000 or http://localhost:3001) and navigate to `/admin/blog-templates`.
